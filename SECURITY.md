# Security Policy

## Project status

Crepitus is an unaudited testnet research project. Do not use it with real funds.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability.

Until a dedicated security mailbox and encrypted reporting process are published, contact the repository owner privately through their verified GitHub profile. Include:

- the affected commit and contract or file;
- a minimal reproduction;
- expected impact;
- whether the issue has been disclosed elsewhere.

No bug bounty is currently offered. This file must be updated before any public production deployment.

## Supported versions

Only the latest commit on `main` is currently maintained. There are no production releases.
